# Code Comprehension

A self-paced course on **reading, debugging, and confidently changing AI-generated code** for web and
AI products — built around *reasoning about code*, not memorizing syntax to write it.

**📖 Read it online:** https://ceciliaw888.github.io/code-comprehension/
*(On iPad: open in Safari → Share → Add to Home Screen to read like an app.)*

## Lessons

| # | Lesson | The one idea |
|---|--------|--------------|
| 01 | [Reading code with confidence](lessons/0001-reading-code-with-confidence.html) | Be the computer — trace code line by line instead of skimming for the gist. |
| 02 | [The shape of a web page](lessons/0002-the-shape-of-a-web-page.html) | A page is a living tree (the DOM); React's job is keeping that tree in sync with your data. |

Plus a [glossary](reference/glossary.html) and an optional
[client/server primer](reference/where-your-code-runs.html).
